About Us
Footprint is a Digital Privacy Dashboard that helps you understand and reduce your online exposure.
Most people don’t realize how often personal information ends up online—through data breaches, reused passwords, and data broker websites. Footprint makes those risks easier to see, so you can take action with confidence.

What is Footprint?

Footprint is a privacy dashboard that checks whether your email or password has been exposed in known breaches, then helps you take the next steps—like securing accounts and removing your data from data broker websites.

What Footprint Can Do

  • Email breach check: shows if your email appears in public breach databases and lists the breached services.
  • Password exposure check: checks whether a password has been seen in known breaches (without sending your raw password).
  • Breach-specific action steps: gives security steps for certain services (example: LinkedIn, Adobe, Dropbox, etc.).
  • Data removal protocol: provides opt-out links and step-by-step instructions for common data broker sites.
  • Website ratings & reviews: lets users rate scanned websites and view community reviews in a dedicated ratings page.

Technology Behind Footprint

Footprint is built with a Flask backend and a JavaScript-powered dashboard. The backend acts as a secure proxy to external privacy APIs (to avoid CORS issues and protect implementation details), and it also stores user activity like opt-out progress and website reviews in the database.

Frontend

JavaScript handles form submissions, loading states, and displaying results. It also powers the removal protocol dashboard and the ratings UI.

Backend

Flask provides authenticated API routes, calls external services for breach checks, and stores reviews + removal progress using SQLAlchemy.

Database

Stores user accounts, removal providers, removal actions, and URL reviews so users can track progress and see community feedback.

APIs We Use

  • XposedOrNot API — checks if an email appears in known breach datasets and returns the breach names.
  • Pwned Passwords (Have I Been Pwned) — checks if a password has been exposed using a privacy-safe method called k-anonymity (only the first 5 characters of the password’s SHA-1 hash are sent).
  • Footprint Internal APIs — removal protocol routes and website review routes powered by our own database.

How Footprint Works

1
Input
You enter an email (and optionally a password) on the dashboard.
2
Secure API Requests
The dashboard sends requests to our Flask backend, which then calls external breach-checking APIs securely server-side.
3
Results + Guidance
You get a clear list of breach results. If breaches are found, Footprint also shows recommended security actions.
4
Removal Protocol
If your email appears in breaches, Footprint loads a data broker opt-out dashboard so you can track “Not started → Submitted → Completed”.
5
Community Reviews
When you scan a website URL, it gets saved locally and you can view/store ratings and comments in the database.
